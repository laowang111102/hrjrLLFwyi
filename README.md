# 学生成绩管理系统 python767

本项目是一个基于Python和Django框架开发的学生成绩管理系统，旨在帮助学校高效管理学生成绩及相关信息。

## 技术栈

- Python
- Django
- HTML/CSS/JavaScript

## 功能模块

### 系统交互

- 登录功能
  - 无需注册，直接使用预设账户登录
  - 管理员用户名：admin，密码固定，不可修改
  - 教师用户名：教师编号，密码默认为教师编号，可修改
  - 学生用户名：学生编号，密码默认为学生编号，可修改

### 修改密码

- 学生、教师可修改个人密码

### 学生模块

- 查询
  - 学生基本信息
  - 该学生所选的各门课程的成绩

### 教师模块

- 查询
  - 教师基本信息
  - 所授课程基本信息
  - 学生选课信息
  - 所授课程所有学生的成绩

## 系统角色

- 管理员
- 教师
- 学生

## 运行环境

- Python 3.x
- Django 2.x

## 部署步骤

1. 安装Python 3.x
2. 安装Django
3. 导入项目数据库
4. 配置环境变量
5. 运行项目
